# MyPortfolio
MyPortfolio is a personal portfolio website designed to showcase skills, projects, and achievements in a clean and professional way. It serves as a digital identity for developers, helping them present their work and connect with recruiters or clients.

## Features
- Modern and responsive design  
- Projects showcase section  
- Skills and experience display  
- Contact section for easy communication
